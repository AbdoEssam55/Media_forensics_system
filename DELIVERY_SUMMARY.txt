## System Capabilities

### Image Detector
- ResNet50 CNN with transfer learning
- Frequency domain analysis (FFT, DCT, entropy)
- Input: PNG, JPG, JPEG, BMP, TIFF
- Output: Real/Fake classification + 0-100% confidence
- Expected accuracy: 94.7%
- Inference: 0.1-0.3 seconds

### Audio Detector
- CNN-LSTM hybrid architecture
- MFCC (13 coefficients) + Mel-spectrogram features
- Spectral analysis (centroid, rolloff, zero-crossing rate)
- Input: WAV, MP3, OGG, FLAC, M4A
- Output: Real/Fake classification + 0-100% confidence
- Expected F1-score: 91.4%
- Inference: 0.2-0.5 seconds

### Video Detector
- Dual independent pipelines:
  * Visual: Frame extraction → CNN inference → temporal consistency analysis
  * Audio: Audio extraction → LSTM analysis
- Scene-agnostic: Works on landscapes, urban scenery, cinematic environments
- No face detection required
- Works on audio-less videos
- Fusion logic: Weighted combination (60% visual, 40% audio)
- Input: MP4, MKV, AVI, MOV
- Output: Per-modality scores + final classification
- Inference: 5-30 seconds per video

## Quick Start Instructions

1. **Install**
   ```bash
   pip install -r requirements.txt
   python setup.py
   ```

2. **Web UI**
   ```bash
   streamlit run app_streamlit.py
   ```

3. **REST API**
   ```bash
   python -m uvicorn app_fastapi:app --reload
   ```

4. **Python Usage**
   ```python
   from media_forensics_complete import ImageDetector
   detector = ImageDetector()
   result = detector.detect('image.jpg')
   print(result['classification'], result['confidence'])
   ```
